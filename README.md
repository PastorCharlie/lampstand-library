# Lampstand Library — Free Christian Resources

> *“Your word is a lamp to my feet and a light to my path.”* — Psalm 119:105

A curated index of free, open-access Christian resources for the body of Christ. Maintained by [Gospel Life Ministries](https://gospellifeministries.org).

## About

The Lampstand Library connects believers with hundreds of thousands of free Christian resources available in the public domain or under open licenses. This repository serves as the master index for the Lampstand Library feature on Discipleship Universe and Gospel Life Ministries.

## Free Resource Repositories

This library draws from the following open-access repositories:

- **[Christian Classics Ethereal Library (CCEL)](https://ccel.org)** — Thousands of classic Christian texts
- **[Project Gutenberg](https://gutenberg.org)** — Public domain books including Christian classics
- **[Internet Archive](https://archive.org)** — Millions of free books with borrowing access
- **[Open Library](https://openlibrary.org)** — Universal library catalog
- **[Desiring God](https://desiringgod.org/books)** — John Piper’s books free online
- **[Monergism](https://monergism.com)** — Reformed theology resources
- **[Bible Gateway](https://biblegateway.com)** — Free Bible translations and study tools
- **[Ligonier Ministries](https://ligonier.org)** — R.C. Sproul’s teaching resources
